# Capacity Planning

Simulation-based multi-period climate-induced resettlement capacity planning under demand and forecast uncertainty.

## Structure

- `Code/`: Python and MATLAB model implementations, simulation code, and generated metrics
- `Figures/`: exported figures
- `Leevi notes/`: presentation and notes

## Main Python Files

- `Code/main_optimize.py`: optimization and scenario analysis workflow
- `Code/simulate_cost.py`: Monte Carlo simulation and cost evaluation
